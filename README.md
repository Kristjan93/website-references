## Animation and motion libraries

The tools behind the magic.

- [GSAP](https://gsap.com/)
  - The gold standard for timeline-based animation. Powers most of the "how did they do that" sites.
  - ScrollTrigger plugin is a game-changer for scroll-driven animations.
- [Framer Motion / Motion](https://motion.dev/)
  - The go-to for React animation. Layout animations and shared element transitions made simple.
  - Rebranded to Motion in 2024 — now framework-agnostic.
- [Lenis](https://lenis.darkroom.engineering/)
  - Smooth scroll library that plays nicely with GSAP and everything else.
  - Used by most award-winning sites for that buttery scroll feel.
- [Three.js](https://threejs.org/)
  - 3D in the browser. Pair with React Three Fiber for a component-based workflow.
- [View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transitions_API)
  - Native browser API for page transitions. No library needed.
  - Supported in Chrome and Safari as of 2025. The future of page transitions on the web.
